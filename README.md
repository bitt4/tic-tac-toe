# tic-tac-toe
Tic-Tac-Toe implemented in C++

## Dependencies
 - SDL2 (with SDL2 gfx extension)

## Options
Use `./tictactoe --help` to see options.

Option          | Effect
--------------- | -------------------------
`--help`, `-h`   | display help
`--size`, `-s`   | set the size of the board

## Installation
```
git clone https://github.com/bitt4/tic-tac-toe.git
cd tic-tac-toe
make
./tic-tac-toe
```
